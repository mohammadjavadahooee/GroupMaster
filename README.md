# ربات مدیریت گروه روبیکا

یک ربات برای مدیریت گروه‌های روبیکا با ویژگی‌هایی مانند قفل کردن محتوا، سیستم هشدار، بی‌صدا کردن کاربران و موارد دیگر.

## ویژگی‌ها
- قفل کردن انواع مختلف محتوا (لینک‌ها، ویدیوها، تصاویر و ...)
- مدیریت هشدارها و مسدودسازی کاربران
- بی‌صدا و بازگشایی کاربران
- مدیریت ادمین‌ها

## نصب
1. مخزن را کلون کنید
2. وابستگی‌ها را با دستور `pip install -r requirements.txt` نصب کنید
3. توکن ربات و شناسه گروه‌ها را در تنظیمات وارد کنید
